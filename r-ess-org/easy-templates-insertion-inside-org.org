To insert a structural element, type a ‘<’, followed by a template selector and <TAB>.

for example, to insert #+BEGIN_EXAMPLE ... #+END_EXAMPLE, type the following 3 keystrokes

| <     |
|       |
| e     |
|       |
| <TAB> |
